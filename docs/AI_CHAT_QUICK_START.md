# AI Chat Interface - Quick Start Guide

## Getting Started

### 1. Open the Chat Interface
Click the **rainbow icon** (✨) in the top-right corner of the window to open the AI chat interface. The drawer will slide in from the right side.

### 2. Select an Agent Mode
Use the **dropdown menu** at the top of the chat to select your desired AI agent mode:
- 🎬 **Screenplay Writer** - Get help writing screenplays
- 📝 **Script Writer** - Write dialogue and scripts
- 📖 **Story Writer** - Develop stories and narratives
- 💬 **Dialogue Writer** - Create character dialogue
- 👤 **Character Developer** - Design and develop characters
- 🌍 **World Builder** - Build fictional worlds

### 3. Type Your Request
Click in the text area at the bottom and type your creative request or question. For example:
- "Write a scene where two characters meet for the first time"
- "Help me develop a protagonist with a tragic backstory"
- "Describe a fantasy kingdom and its culture"

### 4. Send Your Message
- Press **Enter** to send your message instantly
- Or click the **send button** (arrow icon) on the right
- To add a new line without sending, press **Shift + Enter**

### 5. Attach Files (Optional)
To provide additional context:
1. Click the **paperclip icon** 📎 to open file selection
2. Select one or multiple files (.txt, .pdf, .docx, .md, .json)
3. Files will appear as tags above the input area
4. Click the **X** on any file tag to remove it

### 6. Review the Response
The AI will respond with relevant creative writing assistance. Messages are displayed in a chat bubble format:
- Your messages appear on the right with a purple-to-blue gradient
- AI responses appear on the left with a subtle gray background
- Each message shows a timestamp

### 7. Continue the Conversation
Keep chatting with the AI in the same agent mode or switch modes anytime. Your entire conversation history is automatically saved.

### 8. Close the Chat
Click the **X button** in the top-right corner of the chat drawer to close it. Your messages will be saved and available the next time you open the chat.

## Tips & Tricks

### ✨ Best Practices

1. **Be Specific**: The more details you provide, the better the AI can help
   ```
   ❌ "Write a scene"
   ✅ "Write a 2-3 page scene where a detective discovers a clue in a Victorian mansion"
   ```

2. **Use Context**: If continuing from previous writing, paste relevant excerpts
   ```
   "Here's my opening paragraph [paste]. Now help me write the next paragraph."
   ```

3. **Ask Follow-ups**: Refine responses by asking for modifications
   ```
   "Can you make this dialogue sound more like how teenagers talk?"
   ```

4. **Switch Modes for Different Needs**: 
   - Start with "Story Writer" for plot structure
   - Switch to "Character Developer" for character depth
   - Use "Dialogue Writer" for character interactions

5. **Attach Reference Files**:
   - Character descriptions
   - World-building documents
   - Previous chapters or scenes
   - Writing guidelines or style notes

## Message Management

### Viewing History
All conversations are automatically saved to your history. Each conversation is organized by:
- User ID
- Agent Mode
- Timestamp

### Clearing History
To clear your chat history:
1. Close the chat interface
2. Your history remains saved (you can request manual deletion if needed)

## Troubleshooting

### Q: The chat interface won't open
**A:** Make sure you're logged in and click the rainbow icon in the top bar.

### Q: My message isn't sending
**A:** 
- Check that you've typed something in the message box
- Make sure you're not already sending a message (wait for the loading indicator to finish)
- Try refreshing the page

### Q: Files aren't uploading
**A:**
- Check the file format (.txt, .pdf, .docx, .md, .json supported)
- Ensure file size is reasonable (typically <10MB)
- Verify your browser permissions for file access

### Q: The AI response doesn't make sense
**A:**
- Try being more specific with your request
- Provide more context or attach relevant files
- Switch to a different agent mode that might be more relevant
- Try rephrasing your question

### Q: Chat keeps disconnecting
**A:**
- Check your internet connection
- Refresh the page and try again
- Clear browser cache if problems persist
- Contact support if issue continues
