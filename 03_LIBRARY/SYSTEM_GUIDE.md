# System Guide: Folder-Based Writing System v1

## Core Principles

1. **Raw Capture is Frictionless**: Unprocessed creative inputs go to `00_RAW`. No formatting or organization required.
2. **Ideas are Processed**: Viable fragments from `00_RAW` are promoted to `01_IDEAS`.
3. **Projects are Structured by Narrative Unit**: `02_PROJECTS` uses a specific skeleton (Overview, Outline, Drafts by Act/Scene).
4. **Drafts are Split by Act or Chapter**: Avoid giant files; break work into manageable units.
5. **AI Context is Session-Based**: Use `06_AI_CONTEXT.md` in project folders to keep the AI aligned.

## AI Operational Rules

- Ask for missing context before generating long-form material.
- Prefer structured responses over vague inspiration.
- Respect narrative unit boundaries and existing character psychology/canon.
- Do not assume continuity beyond provided text.

## Processing Cycle

Every 3 days:

1. Cull weak ideas.
2. Promote viable ideas to `01_IDEAS`.
3. Optionally expand the strongest idea.
